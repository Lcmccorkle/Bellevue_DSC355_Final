Walmart Weekly Sales Forecaster

This application serves as a practical decision-support tool for inventory planning, staffing, and promotion strategy. While limited to pre-2013 data, it illustrates the value of gradient boosting and feature engineering in real-world time-series forecasting.

Created a Walmart Weekly Sales Forecaster as an interactive web appplicaition on Streamlit and Github.the tool allows users to predict weekly department-level sales for any Walmart store and department by inputting key conditions such as store, size, type, date, holiday status, temperature, fuel price, unemployment rate, CPI and promotional markdown amounts.

Model used: XGBoost regressor trained on historical Walmart sales data with approximately 421,570 observations and 38 engineered features.
